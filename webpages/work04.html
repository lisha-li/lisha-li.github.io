Date: 2015

Type: Information Visualization

Involvement: Research, Design, Program


Description

This is created in Processing and designed for Rouse Hill Town Centre. By using their giftcard data to visualize footsteps of their shoppers, we can find out which is the most popular stores as well as encourage customers to visit various stores by following other's footsteps.

This is a project for DECO3100 - Information Visualization at the University of Sydney.



For more information about this project please look at the documentation.
