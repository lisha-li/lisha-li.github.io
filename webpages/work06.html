Date: 2015

Type: Website/Web Application

Involvement: Project Leader, Research, Development


Description

An interface for customizing and ordering drinks. Additionally it acts as a game/competition where users can try to guess the daily winning formulas for a reward.. It is designed to be used with Kuka a robotic arm that acts as a bartender, mixing and serving drinks. This interface is efficient and easy to set up, just connect it to the local network created by processing to control Kuka.


This is a team project for DECO3200 - Social Robotics at the University of Sydney.
